jQuery.tooltip.js

Directions

  • Top $('a').tooltip({align: 'top'});
  • Left $('a').tooltip({align: 'left'});
  • Right $('a').tooltip({align: 'right'});
  • Bottom $('a').tooltip({align: 'bottom'});

Auto Position

  • AutoTop $('a').tooltip({align: 'autoTop'});
  • AutoLeft $('a').tooltip({align: 'autoLeft'});

If direction is not set, autoTop is used.

Effects

Fade 
$('a').tooltip({fade: true});
Attribute 
$('a').tooltip({attr: 'id'});
Fallback 
$('a').tooltip({falback: 'It was set to nothing!'});
Html 
$('a').tooltip({html: true});
Delay 
$('a').tooltip({delay: {show: 300, hide:300}});

Trigger

show 
$('a').trigger('tooltip:show');
hide 
$('a').trigger('tooltip:hide');
